# Glossary

## Semantic BIM

The interpretation of BIM information beyond geometry, including IFC classes, properties, relationships, intent, evidence and traceability.

## IFC grounding

The explicit linkage between an output and IFC-related classes, properties or relationships.

## Prompt BIM semantic task

A controlled task where a BIM-related prompt is converted into structured, auditable output.

## RESEARCH_PASS

A public research status meaning that the artefact is acceptable as preliminary research evidence. It is not a product-readiness status.

## Conservative governance

A safety-oriented behaviour where the system avoids unsupported technical action when inputs are incomplete or ambiguous.
