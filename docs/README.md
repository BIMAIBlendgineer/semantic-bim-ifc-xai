# docs

## Purpose
This directory contains methodology and contextual documentation explaining the public/private boundaries, experiments, and research positioning of semantic BIM.

## Contents
- `public_boundary.md`: Defines what is public and private.
- `methodology/`: Detailed methodological guides.
- `experiments/`: Summaries of feasibility trials.
- `literature/`: Academic bibliographies and references.
- `assets/`: Image assets and diagrams.
- General research positioning files like `research_overview.md`, `xai_roadmap.md`, and `limitations.md`.

## What this folder does not contain
- It does not contain private datasets or adapter weights.
- It does not contain production plans or certification guarantees.
