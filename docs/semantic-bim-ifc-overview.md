# Semantic BIM/IFC Overview

Semantic BIM/IFC focuses on the meaning of BIM information, not only its geometry.

A semantic BIM/IFC task may involve:

- Identifying the intended IFC class.
- Interpreting properties and quantities.
- Linking a natural-language request to IFC evidence.
- Producing structured outputs.
- Explaining why an answer is supported.
- Tracking missing inputs and uncertainty.
