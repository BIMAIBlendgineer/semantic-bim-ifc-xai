# Methodology

Semantic XAIBIM uses a prompt-to-structure research methodology.

The basic unit is a controlled Semantic BIM case:

1. A BIM-related natural language request.
2. A target semantic interpretation.
3. IFC-related grounding.
4. A structured output.
5. Evidence and traceability fields.
6. Validation metrics.

The initial public package uses a reduced sample and aggregated metrics to support reproducible review without exposing internal training or evaluation workspaces.
