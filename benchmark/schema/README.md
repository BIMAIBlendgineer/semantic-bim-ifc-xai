# benchmark/schema

## Purpose
This folder contains the complete output schema definition for semantic BIM/IFC records.

## Contents
- `semantic_bim_output_schema.json`: Target JSON schema defining output fields for semantic BIM compilation.

## What this folder does not contain
- It does not contain executable validators or libraries.
- It does not contain proprietary IFC schema overrides.
- It does not represent a certified IFC schema checker or regulatory tool.

## Related files
- `sample20/schema_public_sample20_v2.json`: The strict public sample20 v2 contract using JSON Schema Draft 2020-12.
- `docs/methodology/schema_contract_map.md`: Map of the different schemas.
