# Space notes

This Space is a public, reduced and sanitised demonstration of the Semantic XAIBIM harness.

It provides:

- searchable public cases;
- constrained semantic input matching;
- public JSON validation;
- public harness replay over 20 records.

It is not a production system, not a certification tool and not a private model endpoint.

No private model, private dataset, private BIM data, credentials or internal logs are included.
