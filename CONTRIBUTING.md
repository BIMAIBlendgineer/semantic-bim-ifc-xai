# Contributing

Semantic XAIBIM is an open research-oriented repository.

Contributions are welcome in areas such as:

- Semantic BIM/IFC examples.
- Prompt-to-structure benchmark cases.
- Schema validation.
- Explainability patterns.
- Traceability methods.
- Documentation improvements.
- Evaluation metrics.

Do not contribute:

- Private BIM models.
- Proprietary IFC models without permission.
- Protected normative text copied verbatim.
- API keys or credentials.
- Raw internal experiment logs.
- Model adapters or large binary artefacts.
- Commercial deployment claims.
